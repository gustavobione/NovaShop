# Case Digital | NovaShop - Peers Group

## 📌 Sobre o Desafio
Este repositório contém a resolução do case técnico para a posição de Digital Consulting no Peers Group. 

A NovaShop é um e-commerce brasileiro em expansão acelerada, possuindo operações B2C e B2B. Atualmente, a empresa enfrenta uma alta taxa de cancelamentos e devoluções em alguns segmentos, além de um crescimento nos tickets de suporte, sem clareza sobre a causa raiz desses problemas.

## 🎯 Objetivo
Atuando como consultor, o objetivo principal é analisar as bases de dados fornecidas para identificar padrões e anomalias baseadas em evidências que justifiquem a dor atual do cliente[cite: 16].

* **Análise de Dados:** Pandas, Jupyter Notebook
